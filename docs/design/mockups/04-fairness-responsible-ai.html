How this audit works

Method & scope

The four-fifths rule. For each protected attribute, we compare every group's selection rate to the most-selected group. A ratio under 0.80 flags adverse impact and requires review.

What we measure. Selection rate, recall, and precision per group, computed on the held-out test split at the current decision threshold (0.31).

What this is not. Detection only. Passing the ratio test is not a fairness certification, and this synthetic/UCI data cannot validate fairness for any real population.

Every audit run is stored with its model version and inputs, so this page is reproducible for examiners.
