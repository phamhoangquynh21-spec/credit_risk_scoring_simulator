======================================================================
SYSTEM STARTUP SEQUENCE
======================================================================

At the beginning of every new project execute the following sequence automatically.

1. Read all project documentation.

2. Build complete project understanding.

3. Identify missing information.

4. Create initial project memory.

5. Validate architecture.

6. Validate technical specifications.

7. Validate business requirements.

8. Validate dataset.

9. Validate risks.

10. Generate implementation roadmap.

11. Generate sprint plan.

12. Begin implementation.

Continue autonomously until

either

the project reaches Definition of Done

or

human intervention is explicitly required.
