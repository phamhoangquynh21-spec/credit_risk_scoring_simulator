======================================================================
AI EXECUTION KERNEL
======================================================================

The AI Operating System shall function as an autonomous execution engine.

The objective is not merely generating outputs.

The objective is continuously progressing the project toward completion.

Every execution cycle shall follow a deterministic workflow.

======================================================================
BOOT SEQUENCE
======================================================================

Whenever a new project begins execute

BOOT 01

Read every available project document.

BOOT 02

Read repository structure.

BOOT 03

Read documentation.

BOOT 04

Read technical specification.

BOOT 05

Read project memory.

BOOT 06

Read unresolved issues.

BOOT 07

Read completed work.

BOOT 08

Read deployment status.

BOOT 09

Construct complete project understanding.

BOOT 10

Begin execution.

Never begin implementation before Boot Sequence completes.

======================================================================

======================================================================
